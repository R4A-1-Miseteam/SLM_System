# SelfTrack

SelfTrack は、学習の記録と進捗を管理するための学習支援アプリケーションです。

このアプリでは、科目ごとの週次目標を設定し、学習時間やページ数などの実績を入力して、進捗をダッシュボードで確認できます。ToDo 管理や学習データのバックアップ・復元にも対応しており、日々の学習をまとめて追跡できます。

## 主な機能

- 科目ごとの目標設定
- 学習実績の記録
- 進捗の可視化
- ToDo 管理
- JSON 形式でのバックアップと復元

## 開発手順

開発環境での変更を配布するには、以下の手順を実行します。

1. フロントエンドに変更を加える
2. `frontend` フォルダで以下のコマンドを実行してビルドします

```bash
npm run build
```

3. ビルド結果が `selftrack/public` に出力されます
4. その後 `Start-SelfTrack.bat` で起動確認ができます

## 起動方法

### 配布を受け取った側（一般ユーザー）

1. zip ファイルを解凍します
2. ルートフォルダにある `Start-SelfTrack.bat` をダブルクリックします
3. ブラウザが自動で開き、アプリが使用できます

**前提：** PHP 8.0 以上がインストール済みである必要があります

### 開発環境での起動確認

フロント変更後に `npm run build` でビルドしたら、`Start-SelfTrack.bat` をダブルクリックして確認します。

## 配布手順

1. `npm run build` を実行（フロント変更時）
2. SelfTrack フォルダ全体を zip に圧縮
3. 相手に配布
4. 相手は解凍して `Start-SelfTrack.bat` を実行するだけで使用開始
